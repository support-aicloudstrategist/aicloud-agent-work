🔑 KEY OPTIMIZATIONS
═══════════════════════════════════════════════════════════════════════════════

1. OPENROUTER AUTO-ROUTING
   Simple → Gemini 2.5 Flash ($0.075/M, 67x cheaper than Opus)
   Moderate → Deepseek V3.1 ($0.27/M, 18x cheaper)
   Complex → Sonnet 4.5 ($3/M, balanced)
   Critical → Opus 4.6 ($5/M, reserved only)
   Impact: 10x average cost reduction ⚡

2. SESSION AUTO-COMPACTION  
   When context reaches 40K → Summarize & compress to 20K
   Result: 58% reduction in tokens per session
   Impact: Prevents exponential context growth 📉

3. HEARTBEAT OPTIMIZATION
   Before: 48 times/day with expensive model = $36/month
   After: 2 times/day with cheap model = $0.005/month
   Impact: 7,200x reduction (heartbeats alone) 🚀

4. QMD SEARCH
   Load 500KB memory files (15K tokens) → Search locally (5 tokens)
   Impact: 3,000x more efficient memory lookups 🔍

5. AGENT-SPECIFIC MODELS
   Don't use one model for everything
   Assign per task type for optimal cost/performance ratio ⚙️
